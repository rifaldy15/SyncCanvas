"Halo Antigravity, bertindaklah sebagai Senior Full-stack Engineer. Kita akan membangun proyek bernama 'SyncCanvas': sebuah Real-time Collaborative AI Workspace yang 100% berbasis Cloud (Anti-Lokal).

Tujuan Proyek:
Membangun editor dokumen kolaboratif (seperti Notion) yang memiliki fitur AI Assistant menggunakan Groq API dan sinkronisasi real-time menggunakan Supabase.

Tech Stack Spesifik:

Frontend: Next.js 15 (App Router), TypeScript, Tailwind CSS, Shadcn/UI.

Backend/Database: Supabase (Auth & PostgreSQL).

Real-time: Supabase Realtime (Broadcast & Presence).

AI: Groq Cloud SDK (Llama 3).

Animasi: Framer Motion.

Aturan Penulisan Kode:

Gunakan Clean Architecture dengan folder src/components, src/lib, src/services, dan src/app.

Terapkan Server Actions untuk interaksi database.

Pastikan UI mendukung Dark Mode secara default dan sangat responsif.

TUGAS PERTAMA ANDA:

Buat daftar dependensi package.json yang paling relevan untuk stack di atas.

Susun struktur folder proyek yang scalable.

Buat file src/lib/supabase.ts untuk inisialisasi client dan src/lib/groq.ts untuk konfigurasi Groq SDK.

Berikan panduan singkat tentang variabel .env apa saja yang perlu saya siapkan.

Mari kita mulai dengan fondasi infrastruktur ini sebelum masuk ke pembuatan komponen UI."
